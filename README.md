# react-image-mapper2

This repository is based on [react-image-mapper](https://github.com/coldiary/react-image-mapper). Written in Typescript and React FC with some bug fixes.

## Usage

To install dependencies

```
$ yarn install
```

To build

```
$ yarn build
```

To run tests

```
$ yarn test
```
